CSSMin

Install using: 

npm install grunt-contrib-cssmin --save-dev

Example Gruntfile.js: 


...
cssmin: {
  dist: {
    expand: true, // Enable dynamic expansion
    cwd: '_site/assets/', // Where to search for files
    src: ['*.css'], // What files to minify
    dest: '_site/assets/' // Where to save minified files (overwrite)
  }
}
...

This config overwrites existing files.
