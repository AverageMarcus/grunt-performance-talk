Above The Fold

Install using: 

npm install grunt-criticalcss --save-dev

Example Gruntfile.js: 


...
criticalcss: {
  extract: {
      outputfile : './_site/assets/critical.css', // File that will contain above-the-fold css
      filename: './_site/assets/style.css',
      url : 'http://localhost:5000',
      width : 1920, // Target screen width
      height : 1080 // Target screen height
  },
}
...

Note: Above the fold optimization is a topic of debate, see: http://www.iamthefold.com/
