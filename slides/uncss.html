UnCSS

Install using: 

npm install grunt-uncss --save-dev

Example Gruntfile.js: 


...
uncss: {
  dist: {
    options: {
      stylesheets: ['_site/assets/style.css'] // List of stylesheets to clean up
    },
    files: {
      '_site/assets/style.css': ['*.html', '**/*.html'] // What files to check for usage
    }
  }
}
...

This config is a destructive change. Files can be changed to write out to something else.
