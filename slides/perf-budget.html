Perf Budget

Install using: 

npm install grunt-perfbudget --save-dev

Example Gruntfile.js: 


perfbudget: {
  default: {
    options: {
      url: 'http://yourwebsite.com',
      wptInstance: 'http://PRIVATE_INSTANCE.com',
      budget: {
        visualComplete: '3000',
        SpeedIndex: '1000'
      }
    }
  }
}

Note: To have this as part of your dev workflow you'll want to run a privately hosted WebPageTest instance, see: https://sites.google.com/a/webpagetest.org/docs/private-instances
